</Note>

## Command Usage

### `bugster auth`

Connect your CLI to your Bugster account by setting up your API key.

```bash
bugster auth
```

**What it does:**
- Prompts you to enter your Bugster API key
- Validates the API key with Bugster servers  
- Stores authentication credentials securely for future use

## Step-by-Step Setup

<Steps>
  <Step title="Run the auth command">
    ```bash
    bugster auth
    ```
  </Step>
  
  <Step title="Get your API key">
    Find your API key in your [Bugster dashboard](https://dashboard.bugster.dev) under Account Settings.
  </Step>
  
  <Step title="Enter your API key">
    Paste your API key when prompted and press Enter.
  </Step>
</Steps>

## Common Issues

**Invalid API Key:**
- Double-check your API key in the Bugster dashboard
- Make sure you're copying the entire key

**Network Issues:**
- Check your internet connection
- Try again after a few minutes
